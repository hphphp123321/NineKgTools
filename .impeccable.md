## Design Context

### Users
个人用户，管理自己的媒体收藏（音频、视频、游戏、图片、文本）。使用场景偏向个人书房/收藏室式的沉浸式浏览与整理，注重效率和浏览体验。

### Brand Personality
科技感、沉浸感、个人化 — 紫色主题贯穿全局，带有发光效果和玻璃态点缀，营造数字收藏馆的氛围。

### Aesthetic Direction
- **视觉基调**: 深色主题为主，紫色（#8B7FFF）作为品牌色，辅以微妙的发光和渐变
- **参考**: 当前项目已有风格 — frosted glass AppBar、点阵纹理背景、紫色发光滚动条、左侧彩色边框卡片体系
- **反参考**: 不要企业化/严肃风格，不要纯白/扁平设计
- **主题**: 深色主题+白色主题

### Design Principles
1. **一致性优先** — 所有页面遵循统一的卡片体系（card-base, card-bordered-*）、设计令牌和交互模式
2. **桌面优先** — 以桌面体验为核心设计，平板为次要适配目标，手机端保持基本可用
3. **沉浸式浏览** — 海报/封面图是核心视觉元素，交互设计围绕媒体内容的视觉呈现
4. **MudBlazor 原生** — 优先使用 MudBlazor 组件和工具类，自定义 CSS 仅用于补充
5. **渐进增强** — hover 效果、动画、发光等为增强体验，不依赖它们实现核心功能
