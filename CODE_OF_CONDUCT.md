# 贡献者公约（Contributor Covenant Code of Conduct）

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让社区中的每个人都能够免受骚扰，无论其年龄、体型、可见或不可见的残疾、族裔、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人形象、种族、社会阶层、种姓、肤色、宗教或性取向如何。

我们承诺以有助于建立开放、友善、多元、包容、健康社区的方式行事和互动。

## 我们的准则

有助于营造积极氛围的行为包括但不限于：

* 对他人表现出同理心与善意
* 尊重不同的意见、观点和经验
* 给出和得体地接受建设性反馈
* 承担责任，向受我们错误影响的人道歉，并从经验中学习
* 以整个社区的最大利益为出发点，而非仅仅考虑个人

不可接受的行为包括但不限于：

* 使用性化的语言、图像，以及任何形式的性关注或挑逗
* 挑衅、侮辱、贬损性的评论，以及人身或政治攻击
* 公开或私下的骚扰
* 未经明确授权发布他人的私人信息，例如实际地址或电子邮件地址
* 其他在职业环境中被合理认为不恰当的行为

## 执行责任

社区领导者有责任解释和执行我们关于可接受行为的标准，并将对任何他们认为不恰当、威胁、冒犯或有害的行为采取适当且公平的纠正措施。

社区领导者有权并有责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、issue 以及其他贡献，并在适当时说明审查决定的理由。

## 适用范围

本行为准则适用于所有社区空间，并在个人在公共空间正式代表社区时同样适用。代表社区的示例包括使用官方电子邮件地址、通过官方社交媒体账号发布消息，或在线上/线下活动中担任指定代表。

## 执行

辱骂、骚扰或其他不可接受行为可通过以下方式私下报告给维护者：

- **GitHub 私密反馈**：在仓库 [Security 标签页](https://github.com/hphphp123321/NineKgTools/security/advisories/new) 通过 "Report a vulnerability" 入口提交（虽然名为漏洞通道，也可用于行为准则相关举报，仅维护者可见）
- 或者：在 [Issues](https://github.com/hphphp123321/NineKgTools/issues) 提一个**不暴露被举报者身份**的简短 issue，维护者会主动联系

所有投诉都将被迅速且公正地审查和调查；维护者有义务尊重举报人的隐私和安全。

## 执行指南

社区领导者将依照以下社区影响指南来判定他们认为违反本行为准则的行为后果：

### 1. 纠正（Correction）

**社区影响**：使用了不当语言，或其他被认为在社区中不专业或不受欢迎的行为。

**后果**：来自社区领导者的非公开书面警告，阐明违规的性质并解释行为不恰当的原因。可能会被要求公开道歉。

### 2. 警告（Warning）

**社区影响**：一次事件或一系列行为中的违规。

**后果**：对持续行为发出带有后果的警告。在指定时间内禁止与相关人员交互（包括主动与执行行为准则的人员交互）。违反此条款可能导致临时或永久封禁。

### 3. 临时封禁（Temporary Ban）

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间段内禁止与社区进行任何形式的交互或公开沟通。在此期间不允许与相关人员的公开或私下交互，包括主动与执行行为准则的人员交互。违反这些条款可能导致永久封禁。

### 4. 永久封禁（Permanent Ban）

**社区影响**：表现出违反社区标准的行为模式，包括持续的不当行为、骚扰个人，或对某类人群的攻击或贬损。

**后果**：永久禁止在社区内进行任何形式的公开交互。
